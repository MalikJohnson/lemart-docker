Le Mart Returns

Easy, flexible, and hassle-free returns.

Return Policy

  • 90 days for most items with receipt.
  • 365 days for Le Mart brand items with receipt.
  • Final sale items (gift cards, digital downloads, etc.) cannot be returned.

How to Return

Drive-Up Returns

Return items without leaving your car. Start in the Le Mart app.

In-Store

Bring your item to any Le Mart location for assistance.

Mail-In

Use the prepaid label in your order details for free returns.
